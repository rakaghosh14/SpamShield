# SpamShield

SpamShield is an Email Spam Detection System developed using Java Servlet, JSP, JDBC, Oracle Database and Apache Tomcat.

## Features

- Analyze Emails
- Spam Score Detection
- View History
- Delete Records
- Dashboard

## Technologies Used

- Java
- JSP
- Servlet
- JDBC
- Oracle Database 10g XE
- Apache Tomcat 9
- Eclipse IDE
